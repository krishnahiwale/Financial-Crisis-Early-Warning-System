# Financial-Crisis-Early-Warning-System
Real-time financial crisis early warning system using ML models trained on Federal Reserve data — with SHAP explainability, live risk scoring, and WebSocket updates.

## Tech Stack
- Frontend: React 18, Vite, Framer Motion, Recharts, Zustand
- UI: Dark Terminal aesthetic, Custom 3D Canvas
- Data: Simulated FRED API & ML SHAP outputs
